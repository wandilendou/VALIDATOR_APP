**Startup Idea Validator**

An AI powered tool that helps entrepreneurs evaluate and refine their startup ideas using automated scoring and constructive feedback.

**Overview**

This project provides a simple Flask-based web application where users can submit a startup idea and instantly receive:

_A validation score_

Category-level scoring (Market, Problem Clarity, Customer Fit, etc.)

Actionable feedback generated by AI

The goal is to help early-stage founders identify strengths and weaknesses in their ideas before investing time or money.

**Features**

AI-powered scoring system

Clear, structured evaluation categories

Feedback optimized for founders

Simple and responsive UI

Extensible backend (Flask + Python)

Future: Save ideas, compare ideas, export reports, improved analytics

**Tech Stack**

Backend: Python, Flask

AI: OpenAI API

Frontend: HTML, CSS

Database: (Optional) SQLite

Environment: .env for API keys
